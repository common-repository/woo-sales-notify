== Description ==
This is an Extension to WooCommerce Plugin, 
It will promote your recent Sales by showing little notifications on the bottom left of the screen.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/woo-sales-notify` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the WooCommerce->Woo Sales Notify Name screen to configure the plugin
1. Hit Save and generate to initlize the plugin data.
